# Description
Includes the schedule activities required on the project.
For projects that use [[Rolling Wave Planning]] or agile techniques, the activity list will be updated periodically as the project progresses.
## Includes
- Activity identifier
- Scope of work description for each activity in sufficient detail to ensure that project team members understand what work is required to be completed
